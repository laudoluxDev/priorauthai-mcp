# Troubleshooting

## Installer cannot verify the repository

Stop installation and confirm that the source repository is:

```text
https://github.com/laudoluxDev/priorauthai-mcp
```

## Browser login times out

Run the installer again, or use the license-key fallback when prompted.

## Tools do not appear

Restart the MCP client after installation. Claude Desktop, Cursor, and Windsurf
usually load MCP configuration on startup.
